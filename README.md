# Contact Management (AWS Auth Enabled)

This is a minimal React + AWS Cognito authentication starter for your **contact-management**
project. It uses:

- Create React App structure
- React 18 + react-scripts 5
- aws-amplify + @aws-amplify/ui-react
- Cognito Hosted UI via withAuthenticator HOC

## Setup

1. Edit `src/aws-exports.js` and put your real Cognito details.
2. Run:

   npm install
   npm start

3. Deploy with your existing AWS pipeline (Amplify or S3).
